# KPS Glossary

> Canonical terms used across KPS documentation.

## Overview

This glossary standardizes terminology used in KPS documentation.

## Prerequisites

None.

## Main Content

| Term | Definition | Source |
| --- | --- | --- |
| KPS | Koperasi FELDA Debt & Deduction Management system. | PRD-KPS |
| FELDA | Principal stakeholder overseeing cooperative performance and compliance. | PRD-KPS |
| HQ | Central governance scope for multi-site oversight. | PRD-KPS |
| HQ Admin | Global admin with cross-site access to sites, analytics, and settings. | PRD-KPS |
| Site | Operational unit with a unique code and active status. | PRD-KPS |
| Site Admin | Site-scoped role that manages site data and users (if permitted). | PRD-KPS |
| Site Staff | Site-scoped role that executes day-to-day operations. | PRD-KPS |
| Peneroka | Cooperative member whose debt and potongan are managed. | PRD-KPS |
| Hutang | Debt records linked to a peneroka, managed by priority and balance. | PRD-KPS |
| Monthly potongan | Monthly deduction entry per peneroka used for allocation. | PRD-KPS |
| Allocation waterfall | Deterministic priority-based allocation ordering. | PRD-KPS, ARCH-KPS |
| Allocation review | Site workflow for reviewing and adjusting allocations with audit logging. | PRD-KPS |
| Unallocated amount | Remainder stored when potongan exceeds all debts. | PRD-KPS, ARCH-KPS |
| Monthly closing | Locking a site month to prevent edits and record audit data. | PRD-KPS, ARCH-KPS |
| Site context | Active site resolved from route parameter or assignment. | ARCH-KPS |
| Main sidebar | HQ navigation that is always visible to HQ users. | PRD-KPS |
| Site sidebar | Site-scoped navigation shown when a site is selected or for site users. | PRD-KPS |
