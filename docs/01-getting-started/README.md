## Prerequisites

Before you begin, ensure you have:

- PHP 8.2 or higher
- Composer 2.5+
- Node.js 18+
- MySQL 8.0+ or PostgreSQL 14+
- Git

## Recommended Path

1. Start with [Quick Start](01-quick-start.md) to get a feel for the system
2. Follow [Installation Guide](02-installation.md) for proper setup
3. Configure using [Configuration Guide](03-configuration.md)
4. Move to [Development Guide](../03-development/01-developer-guide.md) to start coding
