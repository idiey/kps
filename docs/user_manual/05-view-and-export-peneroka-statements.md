# View and Export Peneroka Statements

> Use `Reports` to review site-level reporting data, open a peneroka statement, and export CSV or
> PDF output.

## Overview

The reporting workspace supports two common tasks in this payment flow:

- export a site-level CSV report
- open and export an individual peneroka statement

For statement handoff, the peneroka statement page is the main destination.

## Open the Reports Workspace

1. Open the site workspace.
2. Click `Reports`.

The report index shows:

- site-level summary cards
- a searchable peneroka list
- debt priority mix
- recent activity
- an `Export Site CSV` action

## Find a Peneroka

Use the search box to find a peneroka by:

- name
- IC number
- phone number

Each row also shows:

- debt count
- outstanding amount
- current month deduction total
- latest deduction month

## Open a Peneroka Statement

1. In the peneroka list, find the correct row.
2. Click `Statement`.

The statement page includes:

- summary cards for debts, outstanding, deductions, allocated, and unallocated totals
- peneroka identity details
- debt summary
- monthly deduction history
- per-deduction allocation rows

## Export Statement Output

From the statement page you can:

- click `Export CSV`
- click `Download PDF`

Use these exports when you need a portable statement outside the browser.

## Export a Site-Level CSV

If you need a broader site report instead of a single statement:

1. Open `Reports`.
2. Click `Export Site CSV`.

This export contains site-level reporting rows derived from the current site report query.

## When Access Is Blocked

The report workflow is separate from deduction entry. If the page is forbidden or missing:

- confirm you are still assigned to the site
- confirm your account has report access
- return to `Potongan Bulanan` or `Allocation Review` only if you are solving an entry issue rather
  than a reporting issue

## Important Scope Note

The current report flow lets you view and export statements. It does not provide a payment approval
or release action from the statement page.
