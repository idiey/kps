## Common Tasks (Web)

### Logging In

1. Navigate to the system URL
2. Enter your email and password
3. Complete verification steps if enabled

### Creating a Job

1. Go to **Jobs**
2. Select **KEW.PA-10** or **Normal** job mode
3. Fill required fields and submit

### Inspection Workflow

- Create inspection reports from the **Inspections** module
- Supervisors/approvers review and approve or reject

### Repair Completion

- Technicians submit completion reports
- Supervisors review and close the job

### Photo Evidence

- Upload before/during/after photos from the **Photos** section
- Toggle photo visibility when needed

### Admin Tasks (Pentadbiran/Company Admin)

- Manage workshops, users, roles, assets, inventory, and settings
- Generate reports (PDF/Excel/CSV)

### Account Settings

- Update profile and password
- Configure two-factor authentication (if enabled)
- Set appearance preferences
