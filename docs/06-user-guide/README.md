## Quick Access by Role

### 🟡 Pentadbiran (Admin Officer)

**Primary Tasks:**

- Receive and register KEW.PA-10 forms
- Create job records in system
- Generate internal KEW.PA-10 forms
- Manage system administration
- Generate reports

**See**: [Admin Officer Guide](02-admin-officer-guide.md)

### 🟣 Penyelia (Supervisor)

**Primary Tasks:**

- Review job requests
- Assign technicians to jobs
- Review completed work
- Quality control
- Close work orders

**See**: [Supervisor Guide](03-supervisor-guide.md)

### 🔵 Pemeriksa (Inspector)

**Primary Tasks:**

- Conduct asset inspections
- Validate asset conditions
- Document findings with photos
- Approve repairs
- Submit inspection reports

**See**: [Inspector Guide](04-inspector-guide.md)

### 🔴 Pelulus (Approver)

**Primary Tasks:**

- Review work orders
- Check budget allocations
- Approve/reject repairs

**See**: [Approver Guide](05-approver-guide.md)

### 🟢 Juruteknik (Technician)

**Primary Tasks:**

- Receive work orders
- Perform repair work
- Document with photos

- Complete work reports

**See**: [Technician Guide](06-technician-guide.md)

## Common Tasks

### Logging In

1. Navigate to system URL
2. Enter government email and password
3. (Optional) Select language preference
4. Click "Log Masuk / Login"

### Switching Language

- Click language toggle (EN/BM) in top right
- All forms and labels update immediately
- Preference saved to user profile

### Uploading Photos

1. Click "Upload Photo" button
2. Take photo with camera OR select from files
3. Add caption/description (optional)
4. Confirm upload
5. Photo appears with timestamp and GPS (if available)

### Digital Signatures

1. Complete all required form fields
2. Click "Sign / Tandatangan"
3. Authenticate with MyKad or government certificate
4. Confirm signature details
5. Submit signed document

## Bilingual Interface

All documentation and system features available in:

- **English (EN)** - Default
- **Bahasa Malaysia (BM)** - Government standard

## Mobile Access

The system is fully responsive for mobile devices:

- Inspections can be conducted offline
- Photos captured directly from mobile camera
- GPS location automatically recorded
- Data syncs when connection restored
